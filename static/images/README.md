# Images Klasörü

Bu klasöre proje görsellerinizi, ekran görüntülerinizi ve diğer resim dosyalarınızı ekleyin.

## Kullanım

Markdown dosyalarınızda şu şekilde kullanabilirsiniz:

```markdown
![Açıklama](/images/dosya-adi.png)
```

## Önerilen Klasör Yapısı

```
images/
├── projects/
│   ├── veilborn/
│   │   ├── screenshot1.png
│   │   ├── screenshot2.png
│   │   └── logo.png
│   └── web-app/
│       ├── homepage.png
│       └── dashboard.png
├── profile/
│   └── avatar.jpg
└── blog/
    └── post-images.png
```

## İpuçları

- Web için optimize edilmiş görseller kullanın (JPG için kalite: 80-90)
- Dosya boyutlarını küçük tutun (ideal: < 500KB)
- Dosya adlarında Türkçe karakter kullanmayın
- Anlamlı dosya adları verin (ornek: `veilborn-gameplay.png`)
